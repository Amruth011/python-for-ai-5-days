# 🚀 Phase 0: Introduction & Getting Started
Before we write our first line of AI-focused Python, we must understand the "Why" and set up a professional developer environment.

### 💡 1. Intro to Programming for AI
Programming is the art of giving precise instructions to a computer. 
* **Traditional Programming:** You write the rules (If/Else) + Input = Output.
* **AI/Machine Learning:** You provide Input + Output = The Machine learns the Rules.

### 🛠️ 2. Professional Environment Setup
A professional tutor doesn't just code in a notepad. We use a standardized stack:
1. **Python Installation:** Download from [python.org](https://www.python.org/). **CRITICAL:** Check the box **"Add Python to PATH"** during installation.
2. **Editor (VS Code):** Install [VS Code](https://code.visualstudio.com/) with the **Python** and **Jupyter** extensions.
3. **Interactive Mode:** We use `Shift + Enter` to run code line-by-line. This is essential for inspecting data tensors and model outputs in real-time.

### 🚦 3. Syntax, Errors & Formatting
* **Python Syntax:** Understanding the "Grammar" of the language.
* **Python Errors:** Learning to read Tracebacks (Syntax vs. Runtime vs. Logic errors).
* **Code Formatting:** Adhering to **PEP 8** standards to ensure your AI pipelines are readable by other engineers.

# 🗓️ Day 01: Python Basics & AI Foundations

Welcome to the first module. Today, we master the essential building blocks that allow AI models to process and store information.

### 🎯 Learning Objectives
* Manage data using **Variables** and appropriate **Data Types**.
* Perform mathematical and logical **Operations**.
* Understand **Control Flow** for decision-making logic.
* Work with core **Data Structures** (Lists, Dicts, Sets) to manage datasets.

---

### 📖 Module 1 Curriculum

#### **Section 1: The Building Blocks**
* **Variables:** Creating labels for your Features and Labels.
* **Comments:** Documenting your AI logic for team collaboration.
* **Data Types:** * **Numbers:** Integers and Floats (Model weights & Probabilities).
    * **Strings:** Foundations for NLP (Natural Language Processing).
    * **Booleans:** Logic switches for decision trees.

#### **Section 2: Working with Data**
* **Operators:** Math calculations (Addition, Multiplication, Powers).
* **String Manipulation:** Cleaning raw text data using Slicing and **f-strings**.

#### **Section 3: Control Flow & Logic**
* **Control Flow:** Designing the "path" your data takes.
* **If Statements:** Conditional logic for AI decision-making.
* **Loops:** Automating repetitive tasks across large arrays and datasets.

#### **Section 4: Data Structures (The AI Essentials)**
* **Data Structures:** Containers for your training data.
* **Lists:** Ordered sequences of data points.
* **Dictionaries:** Key-Value pairs for structured metadata.
* **Tuples & Sets:** Handling immutable and unique data collections.

# 🗓️ Day 02: Functions, Architecture & Classes

**Focus:** Building Reusable and Robust AI Pipelines.



Today we move from writing scripts to building **Modular Systems**. In AI, we rarely write code once; we build functions and classes that can handle thousands of different data inputs.



### 🎯 Learning Objectives

* **Modular Logic:** Wrap complex AI steps into reusable **Functions**.

* **Connectivity:** Learn to work with **External Tools & APIs**.

* **Project Structure:** Organizing files, folders, and **Python Paths**.

* **Robustness:** Implementing **Error Handling** so pipelines don't crash.

* **Scalability:** Using **Object-Oriented Programming (Classes)** for AI agents.



---



### 📖 Module 2 Curriculum



#### **Section 1: Functions (The Engine)**

* **Defining Functions:** Packaging logic for reuse.

* **Parameters & Return Values:** Passing data in and getting predictions out.



#### **Section 2: External Tools & APIs**

* **Import Packages:** Leveraging libraries like `requests` and `numpy`.

* **Working with APIs:** Fetching real-time data for your models.

* **Working with Data:** Processing external files into Python objects.



#### **Section 3: Practical Python & Structure**

* **Project Structure:** Organizing `src/`, `data/`, and `tests/`.

* **Python Paths:** How Python finds your modules across different folders.

* **Working with Files:** Reading/Writing `.csv`, `.json`, and `.txt` files.

* **Organizing Code:** Keeping logic clean and readable.



#### **Section 4: Error Handling (The Safety Net)**

* **Try and Except:** Catching bugs before the program crashes.

* **Common Errors:** Identifying `IndexError`, `KeyError`, and `TypeError` in data.



#### **Section 5: Classes (Object-Oriented AI)**

* **Your First Class:** Blueprints for AI Agents.

* **Methods & Attributes:** Defining what an agent *knows* vs. what it *does*.

* **Inheritance:** Building specialized models from base classes.

* **When to use Classes:** Choosing between functional and object-oriented paths.
